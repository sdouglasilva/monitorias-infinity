O que é o grid?

O CSS Grid Layout é uma técnica poderosa para criar layouts de página complexos de forma mais eficiente e flexível. Ele permite que você divida uma página em uma grade (grid) de linhas e colunas, e coloque itens em diferentes áreas dessa grade.

1- Configurando o Grid

Para começar a usar o Grid Layout, você precisa definir o container que vai usar a grade. Isso é feito aplicando a propriedade display: grid ao elemento container.

Código:

.container {
display: grid;
}

Explicação:

display: grid;
 transforma o elemento com a classe .container em um container de grid. Dentro deste container, você pode definir as linhas e colunas da grid.

2- Definindo Linhas e Colunas

Depois de configurar o container como um grid, você pode definir o número de linhas e colunas usando as propriedades grid-template-rows e grid-template-columns.

Código

.container {
display: grid;
grid-template-rows: 100px 200px;
grid-template-columns: 1fr 2fr;
}

Explicação

grid-template-rows: 100px 200px; define duas linhas com 100 pixels e 200 pixels de altura, respectivamente.
grid-template-columns: 1fr 2fr; define duas colunas. A primeira coluna ocupa 1 fração do espaço disponível e a segunda coluna ocupa 2 frações.

Posicionando Itens na grid

Agora que você definiu a estrutura da grid, pode posicionar os itens dentro dela usando a propriedade grid-area ou as propriedades grid-row e grid-column.

.item1 {
grid-column: 1 / 2;
grid-row: 1 / 3;
}
.item2 {
grid-column: 2 / 3;
grid-row: 1 / 2;
}

Explicação

grid-column: 1 / 2; faz com que o item ocupe da coluna 1 até a coluna 2 (exclusivo).
grid-row: 1 / 3; faz com que o item ocupe da linha 1 até a linha 3 (exclusivo).
No exemplo, .item1 ocupa a primeira coluna e abrange duas linhas, enquanto .item2 ocupa a segunda coluna e a primeira linha.

4- Alinhamento de Itens

Você pode alinhar os itens na grid usando as propriedades align-items, justify-items, align-content, e justify-content.

Código

.container {
display: grid;
grid-template-rows: 100px 200px;
grid-template-columns: 1fr 2fr;
align-items: center;
justify-items: center;
}

Explicação

align-items: center; alinha os itens verticalmente ao centro das suas células
justify-items: center; alinha os itens horizontalmente ao centro das suas células.

Grid Gap

Você pode adicionar espaço entre as linhas e colunas da grid com a propriedade gap.

.container {
display: grid;
grid-template-rows: 100px 200px;
grid-template-columns: 1fr 2fr;
gap: 10px;
}

Explicação

gap: 10px; adiciona 10 pixels de espaço entre todas as linhas e colunas da grid.
